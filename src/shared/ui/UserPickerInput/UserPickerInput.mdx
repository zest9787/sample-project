# UserPickerInput

`UserPickerInput` 컴포넌트는 사용자를 검색하고 선택할 수 있는 입력 필드입니다.

## 주요 기능

- 사번, 이름으로 구성된 입력 필드
- 검색 버튼 클릭 시 사용자 검색 모달 표시
- 모달 내에서 키워드로 사용자 검색
- 검색 결과에서 사용자 선택 및 적용
